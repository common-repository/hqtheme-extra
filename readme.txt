HQTheme Extra adds extras for Marmot theme - Ready sites one click import and hqlib

== Description ==
HQTheme Extra adds extra features to Marmot theme and our plugins. For now it adds one click demo import for Marmot theme. Our common library used by all our products is also part of HQTheme Extra.  In future versions more features will be added.
Learn more on our website: [Marmot.HQWebs.net](https://marmot.hqwebs.net?utm_source=wporg&utm_medium=link&utm_campaign=default&utm_term=extra&utm_content=description)


== Installation ==
Minimum Requirements

1. PHP 7.2 or greater is recommended
2. WordPress 5.5 or greater is recommended


Automatic installation
Automatic installation is the easiest option — WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of HQTheme Extra, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
In the search field type “HQTheme Extra” then click “Search Plugins.” Once you’ve found us, you can view details. Click “Install Now,” and WordPress will take it from there.


Manual installation
Manual installation method requires downloading the HQTheme Extra plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains instructions on how to do this here.

Updating
Automatic updates should work smoothly, but we still recommend you back up your site.


== Frequently Asked Questions ==

Can I use HQTheme Extra without other HQWebS product?
No. HQTheme Extra only contains library for HQWebS products and one click demo install for Marmot theme.


What is Marmot theme?
Marmot is free WordPress theme which provide you full control over your layout with free Elementor page builder.
